# mute-ads

Used to mute applications when ads are playing, for example, applications that sounds like "specify".

## How to use

1. Download [NirCmd](www.nirsoft.net) from their official site.
2. Fill out the config.template.json and save it as `config.json` in the same folder.
3. Run the script (you might have to change your system's scripts execution policies and/or unblock the file).
